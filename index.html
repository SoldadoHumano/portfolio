Meu Trabalho Sobre Mim

Minha Jornada

Apaixonado por Código e Criatividade

Comecei a programar em 2024 e, desde então, venho explorando a tecnologia por conta própria. Gosto de criar pequenos projetos, testar ideias e aprender algo novo a cada experiência.

Meu foco principal tem sido Java, desenvolvendo plugins e soluções para Minecraft com Spigot/Paper e Gradle. Ao mesmo tempo, exploro outras linguagens como C#, C++, Python e JavaScript para ampliar meus conhecimentos e diversificar meus projetos.

Mesmo no início da jornada, vejo cada código que escrevo como uma chance de evoluir e exercitar minha criatividade. Para mim, programar é um espaço onde lógica e imaginação se encontram para transformar ideias em realidade.

Meu objetivo agora é aprender um pouco todos os dias, experimentar novas tecnologias e construir, passo a passo, a base que vai me preparar para projetos maiores no futuro.

Fora do código, os jogos e as lives são meus principais hobbies. Eles me ajudam a relaxar, me divertir e também a compartilhar momentos com outras pessoas. Essa paixão pelos games acaba servindo de inspiração para muitos dos meus projetos.

DeepCraft - Em Desenvolvimento

Este é meu servidor Minecraft Open Source, criado especialmente para a comunidade e também como vitrine dos códigos e sistemas que desenvolvo. Além de servir como um espaço de jogo divertido e colaborativo, ele funciona como demonstração prática das soluções que estou criando, desde plugins personalizados até sistemas de integração, segurança e otimização.

Se você curte tecnologia, desenvolvimento ou simplesmente quer jogar, este projeto é feito para compartilhar conhecimento e também abrir portas para quem quiser adquirir meus sistemas. Quer contribuir, jogar ou saber mais? É só chegar junto!
